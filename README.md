# B2B Client Revenue Analysis

## Project Overview
This project analyzes client revenue and profitability using SQL. The objective is to identify high-value clients, evaluate industry performance, analyze sales channels, and uncover revenue trends.

## Dataset
- 1,500 transactions
- 250 clients
- Multiple industries
- Multiple sales channels

## Tools Used
- SQL
- MySQL

## Key Business Questions
1. Which clients generate the highest revenue?
2. Which industries are most profitable?
3. Which sales channels drive the most revenue?
4. How does revenue change over time?
5. What is the revenue contribution by industry?

## SQL Concepts Used
- GROUP BY
- Aggregate Functions
- CTEs
- Window Functions
- RANK()
- DENSE_RANK()
- LAG()

## Project Files
- SQL_Cleaning.sql
- SQL_Analysis.sql
- Dataset.csv

## Key Insights
- Top clients contribute a significant share of revenue.
- Certain industries outperform others in profitability.
- Revenue trends reveal seasonal patterns.
